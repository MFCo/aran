Aran in Action (better use it with Firefox!)

Welcome the to Aran demonstration page! Aran is a source-to-source code transformer to dynamically instrument JavaScript code. Aran is largely compatible with the ECMAScript5 specification and enables amongst other things: profiling, tracing, sandboxing, and symbolic execution. Here, you may experiment Aran's capabilities by writing JavaScript code both in the master field and in the target field. In the master field, you may assign exports.sandbox, exports.traps and exports.options (see https://github/lachrist/aran for more information). These values will be used to create a new Aran instance after the (Re)Initialize button is cliked upon. This Aran instance can be triggered multiple times on arbitrary target code by clicking on the Run/Analyze button.

Enjoy!
